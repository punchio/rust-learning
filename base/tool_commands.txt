主要工具
rustc，cargo，rustup

rustup命令
rustup update   --更新到最新stable版本
rustup doc    --查看文档
rustup self uninstall --卸载rust

rustc编译
rustc --version   --查看版本号
rustc main.rs   --编译

cargo命令
cargo --version   --查看版本号
cargo new project_name    --创建工程
cargo build   --编译整个工程
cargo run   --执行文件
cargo check   --检查编译错误不生成执行文件，比build快
